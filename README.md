Velvet & Vogue | Summer Collection 👗
A sophisticated and minimalist fashion landing page that redefines elegance through a clean, modern interface. This project focuses on high-end visual storytelling and editorial-style layouts.

🚀 Key Features
3-Column Grid Layout: A structured and balanced design that highlights both imagery and core content.

Editorial Typography: Uses 'Bodoni Moda' for a luxury fashion magazine feel and 'Montserrat' for professional readability.

Interactive Visuals: Grayscale-to-color hover effects on images and smooth scaling transitions.

Minimalist Aesthetic: Clean white space and decorative elements like the vertical divider line to enhance the premium feel.

🛠️ Technologies Used
HTML5: Structured with semantic elements for a modern web experience.

CSS3: Advanced CSS Grid, transitions, and grayscale filters for interactive design.

Google Fonts: Bodoni Moda (Luxury Serif) and Montserrat (Sleek Sans-Serif).
